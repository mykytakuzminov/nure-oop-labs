## Objective

Study of the core features of the C++ language, including class creation and usage.

## Tasks

Design and implement a "Number Queue" class based on a dynamic linked list.

Class data: pointer to the front of the queue, pointer to the back of the queue.

Class functions:
- Peek — read the front element without removal
- Dequeue — read and remove the front element  
- Enqueue — add an element to the back of the queue

Required interface:
- All types of constructors with data validation
- Access and initialization functions
- Equality comparison function
- Console input and output functions
